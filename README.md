# Car Suggestion Website

## About the Project
This project is developed as part of my academic work.  
The Car Suggestion Website helps users select a suitable car based on their budget, usage, and preferences.

The main goal of this project is to apply web development concepts to solve a real-life problem.

---

## Objectives
- To help users choose the right car easily
- To provide clear car comparisons
- To display important car details in one place
- To improve decision making for car buyers

---

## Features
- Car suggestions based on user input
- Search and filter options
- Car comparison functionality
- Detailed car information
- Simple price and EMI calculation
- Easy to use interface

---

## Technologies Used
- HTML
- CSS
- JavaScript
- JSON for car data
